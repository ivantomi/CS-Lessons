How-to Guide: Creating a Form

Forms are a great way to collect information from users. Let's go over how to create a form using HTML and CSS.

First things first, let's go over the elements we'll be using:

  • <form> - This element is used to create a form. Everything inside of this element will be part of the form.
  • <input> - This element is used to create an input field. There are many different types of input fields, such as text, number, and dropdowns. Also, they're used to create radio buttons (single choice) and checkboxes (multiple choice).
  • <select> - This element is used to create a dropdown menu.
  • <button> - This element is used to create a button.
  • <label> - This element is used to create a label for an input field.
  • <fieldset> - This element is used to group related elements together.
  • <legend> - This element is used to create a title for a fieldset.

Basic HTML

Let's start by creating a basic form with a text input, number input, dropdown, radio buttons, checkboxes, and a submit button.

First, we'll create a form element and give it an id of "form". Inside of the form, we'll create a text input and give it an id of "text-input". We'll also create a number input and give it an id of "number-input". Next, we'll create a dropdown and give it an id of "dropdown". Inside of the dropdown, we'll create three options. We'll also create a fieldset with three radio buttons and a fieldset with three checkboxes. Finally, we'll create a submit button.

Here's what our code looks like so far:

          
            <form id="form">
              <input type="text" id="text-input" />
              <input type="number" id="number-input" />
              <select id="dropdown">
                <option>Option 1</option>
                <option>Option 2</option>
                <option>Option 3</option>
              </select>
              <fieldset>
                <legend>Radio Buttons:</legend>
                <label>Option 1</label><input type="radio" />
                <label>Option 2</label><input type="radio" />
                <label>Option 3</label><input type="radio" />
              </fieldset>
              <fieldset>
                <legend>Checkboxes:</legend>
                <label><input type="checkbox" /> Checkbox 1</label>
                <label><input type="checkbox" /> Checkbox 2</label>
                <label><input type="checkbox" /> Checkbox 3</label>
              </fieldset>
              <button>Submit</button>
            </form>

What's up with those "types"?

The "type" attribute is used to specify what type of input field you want to create. For example, if you want to create a text input, you would use type="text". If you want to create a number input, you would use type="number". If you want to create a dropdown, you would use type="dropdown". If you want to create a radio button, you would use type="radio". If you want to create a checkbox, you would use type="checkbox". That way, there's only one component we need - input.

Another useful feature we're using is "fieldset". Fieldsets are used to group related elements together. For example, if you have a form with a bunch of checkboxes, you can group them together using a fieldset. This makes it easier for users to understand what the checkboxes are for. Additionally, it allows us to add a title to the group of checkboxes using "legend". It example above, there is a fieldset for the radio and checkbox buttons.

Labels and submitting

Now that we have a basic form, let's add some labels and styling.

Labels are used so that the user understands what each input field is. Even though you could do it with regular text, it's better to use labels because it makes it easier for screen readers to read the form. To create a label, you would use the "label" element. For example, if you want to create a label for a text input, you would use <label>Text Input:</label>. The text inside of the label is what will be displayed to the user.

Here's what our code looks like so far:

          
            <form id="form">
              <label>Text Input:</label>
              <input type="text" id="text-input" />
              <label>Number Input:</label>
              <input type="number" id="number-input" />
              <label>Dropdown:</label>
              <select id="dropdown">
                <option>Option 1</option>
                <option>Option 2</option>
                <option>Option 3</option>
              </select>
              <fieldset>
                <legend>Radio Buttons:</legend>
                <label>Option 1</label><input type="radio" />
                <label>Option 2</label><input type="radio" />
                <label>Option 3</label><input type="radio" />
              </fieldset>
              <fieldset>
                <legend>Checkboxes:</legend>
                <label><input type="checkbox" /> Checkbox 1</label>
                <label><input type="checkbox" /> Checkbox 2</label>
                <label><input type="checkbox" /> Checkbox 3</label>
              </fieldset>
              <button>Submit</button>
            </form>

Finally, now that we have our form, let's see how we can use it as it's supposed to be used in the next step.

Required and submitting

Now that we have a form, let's see how we can use it as it's supposed to be used.

First, let's add the "required" attribute to each input field. This will make it so that the user has to fill out each input field before they can submit the form. To add the "required" attribute, you would use required. For example, if you want to make a text input required, you would use <input type="text" required />.

Next, let's add a submit button. This will allow the user to submit the form. To create a submit button, you would use <button>Submit</button>. Notice what happens when you fill in all the required data and click the button "Submit".

Here's what our code looks like so far:

          
            <form id="form">
              <label>Text Input:</label>
              <input type="text" id="text-input" required />
              <label>Number Input:</label>
              <input type="number" id="number-input" required />
              <label>Dropdown:</label>
              <select id="dropdown" required>
                <option>Option 1</option>
                <option>Option 2</option>
                <option>Option 3</option>
              </select>
              <fieldset>
                <legend>Radio Buttons:</legend>
                <label>Option 1</label><input type="radio" />
                <label>Option 2</label><input type="radio" />
                <label>Option 3</label><input type="radio" />
              </fieldset>
              <fieldset>
                <legend>Checkboxes:</legend>
                <label><input type="checkbox" /> Checkbox 1</label>
                <label><input type="checkbox" /> Checkbox 2</label>
                <label><input type="checkbox" /> Checkbox 3</label>
              </fieldset>
              <button>Submit</button>
            </form>
