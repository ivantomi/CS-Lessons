Class selectors

Class selectors are used to apply the same styling to multiple elements. They are referenced in CSS using a dot followed by the name of the class. For example, if we want to apply the same styling to all the elements with the class "example", we would use the following CSS:

.example {
     /* CSS styling goes here */
}

Nesting selectors

Selectors can be nested inside other selectors. For example, if we want to apply the same styling to all the elements with the class "example" that are inside a div, we would use the following CSS:

div .example {
     /* CSS styling goes here */
}

:hover pseudoclass

The :hover pseudoclass is used to apply styling to an element when the mouse is over it. For example, if we want to change the background color of all the elements with the class "example" when the mouse is over them, we would use the following CSS:

.example:hover {
     /* CSS styling goes here */
}
