Step 1: Set up your environment

First, you need to set up your environment. This means you need to install the tools you will use to create your website. The tools you need are:

  • Visual Studio Code
  • A web browser

Once you do that, here's what you do:

  1. Open VS Code
  2. Open/Create a folder
  3. Create a file called index.html
  4. Open the file in your browser

Step 2: Basic Elements

Headings

As in Google Docs, there are multiple headings you can add to HTML, ranging from 1 to 6

To add one, type the following code:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Paragraphs

Paragraphs are used to add text to your website. To add one, type the following code:

<p>This is a paragraph</p>

Links

Links are used to link to other pages. To add one, type the following code:

<a href="https://www.google.com">This is a link</a>

The thing that actually allows you to redirect is the href attribute, while everything that's inside the tags is what's going to be shown on the page itself (not necessarily text, you can nest whatever you want, like images for example). Don't forget to include the whole link (with https), otherwise it won't work.

Images

Img tags are used to add images to your website. To add one, type the following code:

<img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png"

The thing that actually allows you to add an image is the src attribute. Don't forget to include the whole link (with https), otherwise it won't work.

Lists

Lists are used to add lists to your website. To add one, type the following code:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Also, you can automatically create an ordered list by changing it from "ul" (unordered list) to "ol" (ordered list), like so:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Divs

Divs are used to group elements together. To add one, type the following code:

<div>This is a div</div>

Here's an example of the way you can use divs with paragraphs and images:

<div>
    <p>This is a paragraph</p>
    <img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png"
</div>

Don't forget - The div will automatically take up the whole width of the page.

Step 3: Basic Styling

Importing CSS

Now that we know how to add elements to our website, we can start styling them. There are multiple ways to style elements, but the most common one is using CSS. CSS stands for Cascading Style Sheets, and it's used to style HTML elements. To add CSS to your website, you need to create a file called "style.css" and link it to your HTML file. To do that, type the following code inside the head tag:

<link rel="stylesheet" href="style.css">

Selectors

Now, let's see how we can style elements. Let's say we want to change the color of all the paragraphs on our website to red. To do that, we need to add the following code to our style.css file:

p {
    color: red;
}

The "p" in the code above is called a selector. It's used to select the elements you want to style. In this case, we selected all the paragraphs on our website. The code inside the curly brackets is called a declaration. It's used to declare the style of the selected elements. In this case, we declared that the color of all the paragraphs on our website should be red. Now, let's see how we can style a specific element. Let's say we want to change the color of a specific paragraph to blue. To do that, we need to add the following code to our style.css file:

#blue-paragraph {
    color: blue;
}

In this case, we used an id selector to select the paragraph we want to style. The id selector is used to select a specific element. In this case, we selected the paragraph with the id "blue-paragraph". Now, let's see how we can style multiple elements at once. Let's say we want to change just some of the paragraphs on our website to green. To do that, we need to add the following code to our style.css file:

.green-paragraph {
    color: green;
}

In this case, we used a class selector to select the paragraphs we want to style. The class selector is used to select multiple elements. In this case, we selected all the paragraphs with the class "green-paragraph".

Common Styling Attributes

Here are some of the common attributes you can change on elements:

  • color
  • background-color
  • font-size
  • font-family
  • text-align
  • width
  • height
  • border
  • border-radius
  • padding
  • margin

Step 4: Positioning (Flexbox)

Setting up Flexbox

Let's start by creating a basic div element with multiple squares inside it:

<div class="container">
    <div class="square">1</div>
    <div class="square">2</div>
    <div class="square">3</div>
    <div class="square">4</div>
    <div class="square">5</div>
</div>

In order to be able to position the squares, we need to add the following code to our style.css file:

.container {
    display: flex;
}

Changing the direction

Now, let's say we want to position the squares in a row. To do that, we need to add the following code to our style.css file:

.container {
    display: flex;
    flex-direction: row;
}

Now, let's say we want to position the squares in a column. To do that, we need to add the following code to our style.css file:

.container {
    display: flex;
    flex-direction: column;
}

Changing the alignment

Now, let's say we want to position the squares in a row, but we want them to be positioned in the center of the container. To do that, we need to add the following code to our style.css file:

.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
}

Justify content is used to distribute the elements alongside the main axis (if it's a row horizontally, if it's a column vertically). If we want to change the positioning of the squares along the cross axis (if it's a row vertically, if it's a column horizontally), we need to add the following code to our style.css file:

.container {
    display: flex;
    flex-direction: row;
    align-items: center;
}

Other options instead of center are:

  • flex-start
  • flex-end
  • space-between
  • space-around
