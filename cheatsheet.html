HTML Elements

  • a - Defines a hyperlink
  • br - Inserts a single line break
  • div - Defines a section in a document
  • h1 - Defines a large heading
  • img - Defines an image
  • p - Defines a paragraph

CSS Properties

  • background-color - Sets the background color of an element
  • border - Sets all the border properties in one declaration
  • color - Sets the color of text
  • display - Specifies the type of box used for an HTML element
  • flex-direction - Specifies the direction of the flexible items
  • height - Sets the height of an element
  • justify-content - Specifies the alignment between the items inside a flexible container when the items do not use all available space
  • margin - Sets all the margin properties in one declaration
  • padding - Sets all the padding properties in one declaration
  • text-align - Specifies the horizontal alignment of text in an element
  • width - Sets the width of an element

Classes and IDs

Classes and IDs are used to give elements a name, so that they can be styled using CSS. Classes are used to style multiple elements, while IDs are used to style a single element.

Classes are defined using the class attribute, and IDs are defined using the id attribute. Both attributes can be used on any HTML element.
